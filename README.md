# Проект algorithms-and-ds

### Цель проекта

Проект создан с целью изучения и реализации базовых алгоритмов и структур данных на python; изучения фреймворка pytest.

### Описание

* Алгоритмы
    - Insertion sort
    - Selection sort
    - Heap sort
    - Merge sort
    - Quick sort

* Структуры данных
    - Stack
    - Queue
    - Tree
    - Heap
