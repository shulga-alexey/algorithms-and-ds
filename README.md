# Проект algorithms-and-ds

### Цель проекта

Проект создан с целью изучения, реализации и тестирования
базовых алгоритмов и структур данных на Python.

### Описание

* Алгоритмы
    - Insertion sort
    - Selection sort
    - Heap sort
    - Merge sort
    - Quick sort

* Структуры данных
    - Stack
    - Queue
    - Tree
    - Heap
